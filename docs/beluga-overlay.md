
## Beluga Overlay (webpages)

There is also an optional extension at [`beluga-overlay/`](beluga-overlay/) that injects the whale onto **webpages** as a top overlay (it cannot inject into Chrome’s actual toolbar UI).

- Install: Chrome → `chrome://extensions` → Developer mode → Load unpacked → select `beluga-overlay/`.
- Icons: a GitHub Action auto-renders PNG icons from `beluga-overlay/whale.svg` because Chrome does not support SVG icons in manifests.
